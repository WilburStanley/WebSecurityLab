# WebSecurityLab
Web security scripts for educational purposes in a controlled lab environment

## Disclaimer
All scripts in this repository were created strictly for
educational purposes within a controlled lab environment.
Do NOT use these scripts against any real systems without
explicit written permission.

## About
This repository contains scripts focused on web security
testing and automation. Each folder contains scripts for
a specific topic along with its own README explaining
usage and assumptions.

## Requirements & Limitations
Varies per script. Check the README inside each folder
for specific requirements.
